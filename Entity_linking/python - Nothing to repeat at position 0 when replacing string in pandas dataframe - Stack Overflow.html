+ is a special character and you have to escape it with \:

import pandas as pd
data5 = pd.DataFrame({"id": [1, 2, 3], 
                      "phone": ["+62634783472", "0834838483", "+62"]})

data5['phone'] = data5['phone'].str.replace('\+62', '0')

# data5
   id       phone
0   1  0634783472
1   2  0834838483
2   3           0
